# XIV Crafting Helper

An application that presses your crafting macros for you.

### Screenshot

![App](https://github.com/bronchysaurus/xiv_crafting_helper/raw/main/screenshots/app.png "app")

### Installation

**Prerequisties**

Python 3

1. Clone the application

```sh
git clone https://github.com/bronchysaurus/xiv_crafting_helper.git
```

2. Activate the virtual env and run the application

```sh
cd xiv_crafting_helper
```

```sh
.\w-venv\Scripts\activate
python3 .\main.py
```
